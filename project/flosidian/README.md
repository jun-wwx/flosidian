# NeuroNote

A local-first bidirectional note-taking app built with Tauri + React.

## Features (MVP v0.1)

- 📝 Markdown Editor with live preview
- 🔗 Bidirectional Links (`[[wiki-style links]]`)
- 📎 Backlinks panel
- 💾 Auto-save

## Tech Stack

- **Frontend:** React + TypeScript + Vite
- **Backend:** Tauri (Rust)
- **Storage:** Local Markdown files

## Development

```bash
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri build
```
